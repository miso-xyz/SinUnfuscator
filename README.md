# SinUnfuscator
Basic Deobfuscator for SinUnfuscator

# Protection Supported
<pre>
Supported:
- String Encryption
- Anti-Debug
- Anti-de4dot
- Junk
- Anti-dump

Not Supported:
- ControlFlow
- Renamer (the renaming is random, you cannot reverse the original name)
</pre>

# Usage
Just Drag & Drop the protected file to clean it
